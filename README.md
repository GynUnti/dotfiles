# Introduction
This is the place for me to store all of my .config files for my **EndeavourOS** machine.
This directory contains many config files for different applications, such as Neovim, i3 window manager, kitty terminal, Rofi, etc.
Some of them were shipped by default, but I've changed many of them to fit my preference.

# What you can do with it
You can clone the directory and put these config files to your .config directory to try out some of my configs.
```bash
git clone https://github.com/GynUnti/dotfiles.git
```
copy/move the config file to your .config folder
```bash
cp dotfiles/[your desired config] ~/.config
```
